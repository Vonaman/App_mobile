Chaise Hub mobile c'est :
- une application mobile directement connecté au site chaiseHub
- un outil simple et efficace de visualisation de chaise sur mobile
- mais surtout... un comparateur de chaise !
- vous hésitez entre deux chaises ? Comparer les !
- avec une API partagé, connectez votre compte du site pour mettre au panier directement sur votre téléphone/tablette !
- Peut etre que les utilisateurs pourront noté les chaises, notes qui seront directement rappatrié sur le site internet
